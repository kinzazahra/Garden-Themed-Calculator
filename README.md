Garden Calculator – Nature-Inspired Math
A refreshing, garden-themed web application that brings a touch of nature to your daily calculations. This interactive tool combines essential mathematical functionality with a vibrant, eco-friendly aesthetic, featuring blooming animations and a serene user interface.

Features
Nature-Inspired UI:

Eco-Friendly Palette: A beautiful interface using a blend of nature-inspired colors like Fresh Leaf Green (#7AB2B2) and Soft Floral Pink (#EEA5A6).

Interactive Foliage: Hover over buttons to see them bloom with a subtle green glow, making math feel like a walk in the park.

Full Calculation Suite:

Basic Arithmetic: Handle addition, subtraction, multiplication, and division with ease.

Advanced Operations: Includes support for percentages and square roots, perfect for both quick sums and more complex garden planning.

Interactive Feedback:

Floating Petals: Features a dynamic background with floating animated petals to create a calming, immersive atmosphere.

Clear Display: A large, easy-to-read numeric display housed in a "Glassmorphism" styled card.

Responsive & Modern:

Built using a single-file architecture (HTML/CSS/JS) for lightning-fast loading and zero dependencies.

Fully responsive layout that scales beautifully from large desktop monitors to mobile screens.

🛠 Tech Stack
Structure: HTML5.

Styling: CSS3 (Flexbox, Grid, and Keyframe Animations).

Logic: Vanilla JavaScript (ES6+) for mathematical processing and DOM manipulation.

⚙️ Installation & Setup
Since this is a lightweight frontend application, no installation is required!

Clone the Repository:

Bash
git clone <your-repo-url>
cd Garden-Themed-Calculator
Run the Application:

Simply locate garden-calculator.html in your file explorer and double-click it to open in your preferred browser.

Alternatively, use the Live Server extension in VS Code for the best experience.

🚀 How to Use
Calculate: Use the numeric pad to enter your values just like a standard calculator.

Clear Systems: Use the AC (All Clear) button to reset your workspace or the DEL button to fix minor mistakes.
